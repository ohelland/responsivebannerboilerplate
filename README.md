# responsivebannerboilerplate
Boilerplate for developing responsive banners

## Getting started
 * Install node
 * ```npm install -g gulp ```
 * ``` npm install ```
 * ``` gulp server ```
 * Browse [http://localhost:8888/preview.html](http://localhost:8888/preview.html)

## Building dist
 * ``` gulp build ```
 * Zip file is located in dist folder
